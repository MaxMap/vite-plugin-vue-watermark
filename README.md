# vite-plugn-vue-watermark

> vite3 + vue3 + rollup 实现水印效果



## 使用方法

```js
import watermark from 'vite-plugin-vue-tool'
```
## 使用参数

|  属性   | 含义  | 默认值  |
|  ----  | ----  | ----  |
| text  | 内容 | "我是水印"  |
| width  | 宽度（px） | 120  |
| height  | 高度(px) | 100  |
| angle  | 角度 | -40  |
| font  | 像素大小以及字体 | "15px 黑体"  |
| color  | 颜色 | 'rgba(100,100,100,0.2)'  |
